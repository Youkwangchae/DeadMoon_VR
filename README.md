### DeadMoon_VR
2023년 2학기 건국대학교 컴퓨터공학부 게임프로그래밍 과목을 통해 제작한 VR 공포 게임입니다.

### Target Device
Oculus Quest 2

### Game Concept
- Free Movement with Arm-Shaking
- Puzzle Combination
- Japanese Cultural Vibe
